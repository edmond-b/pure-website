JavaScript

Run Boris, Run is a Super Mario style side scroller that I worked on as my final project for Makers Academy. It was completed in less that two weeks using only vanilla JavaScript for the game - that means we had to code in everything from physics to animations - and React for the site layout.
I'm very proud of this as it is the first game that I've deployed for the public to play. You can play the game at the link bellow and check out the GitHub page to see some of the tech used.
